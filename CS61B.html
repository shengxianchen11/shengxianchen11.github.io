Signpost

The puzzle game Signpost is one of Simon Tatham's collection of GUI games (available on a variety of operating-system configurations, including Ubuntu.) He attributes the puzzle to Angela and Otto Janko, who call it Pfeilpfad (arrow path). In this project, we are given an incomplete Java program that creates these puzzles and allows its user to solve them, and we must supply the missing parts to complete the implementation.

Line of Actions

Brief description of LOA

Lines of Action is a board game invented by Claude Soucie. It is played on a checkerboard with ordinary checkers pieces. I implemented this program which respond to different textual commands, which is able to determine the game state and which side wins. I also implemented an AI for which maintain a 60& - 70% win rate against human players, and which will be able to find forced wins within small number of moves

Gitlet, your own version-control system

In this project we implemented a version-control system that mimics some of the basic features of the popular system Git. Ours is smaller and simpler, however, so we have named it Gitlet.

I implemented these functionality of this version-control system.

  • Saving the contents of entire directories of files. In Gitlet, this is called committing, and the saved contents themselves are called commits.
  • Restoring a version of one or more files or entire commits. In Gitlet, this is called checking out those files or that commit.
  • Viewing the history of our backups. In Gitlet, we view this history in something called the log.
  • Maintaining related sequences of commits, called branches.
  • Merging changes made in one branch into another.
